Résumé :

Les balises en HTML servent à indiquer au navigateur (et parfois au moteur de recherche) ce à quoi correspond le texte entre les balises. Est-ce juste un texte ? Un texte à mettre en rouge ? En gras ? Une image à afficher ? Un lien ?...

Certaines balises seront déjà interprétées par le navigateur selon certaines règles prédéfinies incluses dans le navigateur (par exemple : afficher les liens en bleu), mais pour modifier ces règles ou définir d'autres règles générales (par exemple : afficher plutôt les liens en rose), ça se fait de préférence dans un fichier CSS.

Si on définit en CSS : afficher le contenu des balises img avec une hauteur 100px, alors ça devient une règle qui va s'appliquer directement à toutes les balises img du fichier HTML (inutile de toutes les régler individuellement à la main). Puis si l'on veut ne sélectionner que certaines balises img, on ajoute une class au sein de la balise pour pouvoir cibler uniquement les balises img appartenant à cette classe. Par exemple je veux que certaines images s'affichent à gauche de la page et d'autres à droite. Dans les balises img j'ai juste à ajouter class="gauche" ou class="droite", puis dans le fichier CSS je définis la règle : tout ce qui est de classe "gauche" il faut l'afficher à gauche, et ce qui est de classe "droite" l'afficher à droite.

Autre exemple, je veux afficher certains mots importants dans un texte en rouge souligné. Il suffit que je donne à ces mots une classe, par exemple la classe "important". Je définis cette classe dans le fichier CSS : tout ce qui est "important", le mettre en rouge et le souligner. Puis pour attribuer cette classe "important" à ces mots ou groupes de mots, il faut que je les place entre balises (une classe se met à l'intérieur d'une balise). Si je n'ai pas de type de balise spécifique à utiliser j'utilise une balise générique (span ou div). Ici c'est pour un élément en ligne (inline) alors c'est span que j'utilise (div c'est pour un élément en bloc). Le texte que je veux écrire : "Dans ce texte ce mot est très important. Et cette phrase aussi.".

En HTML je l'écris ainsi :

<p>Dans ce texte ce <span class="important">mot</span> est très important. <span class="important">Et cette phrase aussi.</span></p>

➔ Dans ce texte ce mot est très important. Et cette phrase aussi.
